# Unit Converter Utility

This directory contains the source code for a robust unit conversion utility application. Designed to perform precise mathematical transformations across various measurement systems, the application focuses on high-performance logic execution and strict real-time input validation. The native Android UI is constructed using optimized XML layouts to handle multiple screen densities and orientations seamlessly. 

The application utilizes internal state preservation techniques to ensure that user inputs, selected conversion metrics, and calculated results are maintained accurately during Android Activity lifecycle events, such as device rotation. The underlying conversion logic is strictly separated from the UI layer, promoting a clean architectural pattern that allows for easy maintainability and the future addition of complex measurement categories.

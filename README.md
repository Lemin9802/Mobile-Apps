# Native Android Applications Portfolio

This repository contains a curated portfolio of native Android applications developed using Android Studio. The collection demonstrates a clear progression of mobile software engineering skills, ranging from fundamental utility applications and local state management to complex, dual-client enterprise ecosystems with real-time cloud synchronization. The repository is organized into three primary projects: a comprehensive Yoga Class Management system, a Unit Converter utility, and a scalable To-Do List application.

Throughout these projects, development heavily emphasized robust architecture, native UI/UX design using XML layouts, and efficient data persistence strategies. The applications leverage the Android Activity and Fragment lifecycles, structured intent-based navigation, and SQLite for offline-first data management. For the advanced projects, bidirectional data synchronization with Firebase was implemented to ensure data integrity and real-time availability across multiple clients. Strict input validation methodologies were consistently applied to maintain application stability and a seamless user experience.

To explore the source code or test any of these applications locally, clone this repository and open the specific project directory directly in Android Studio. Ensure that your Android SDK and Gradle build tools are properly configured. Once the Gradle synchronization is complete, compile and deploy the applications onto an Android Virtual Device (AVD) or a physical Android testing device via USB debugging.

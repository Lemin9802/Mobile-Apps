# Yoga Class Management Ecosystem

The Yoga Class Management project is a comprehensive, dual-application solution designed to digitize studio operations and client bookings. Built natively for Android, the ecosystem relies on a shared Firebase backend architecture to synchronize data in real-time between the administrative and customer-facing applications. Both applications utilize a robust offline-first strategy, leveraging local SQLite databases to cache structural data and ensure core functionalities remain accessible during network interruptions. The underlying data flow dictates that all course schedules and class instances must pass rigorous schema validation before being committed to the database.

The Admin Portal serves as the primary operational tool for studio managers. It handles complex data entry workflows, allowing administrators to perform full Create, Read, Update, and Delete (CRUD) operations on yoga courses and specific class instances. A critical technical feature of this administrative app is its background synchronization module, which securely monitors local SQLite changes and pushes these updates directly to Firebase whenever network connectivity is restored, ensuring the master schedule is consistently accurate across the cloud.

The Customer Portal acts as the client-facing frontend, providing end-users with an intuitive native experience for discovering and booking available classes. This application dynamically fetches real-time schedules from Firebase, populating custom-built RecyclerViews for efficient data display. It features advanced client-side filtering algorithms, enabling users to query classes by instructor, class type, or schedule. Efficient state management techniques are utilized to maintain the user's booking cart across different activities prior to the final transaction confirmation.
